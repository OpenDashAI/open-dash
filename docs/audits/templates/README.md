# Review Templates

Reusable templates for conducting different types of reviews.

## Files

| Document | Review Type | Use Case |
|----------|-------------|----------|
| [TECHNICAL-REVIEW-TEMPLATE.md](./TECHNICAL-REVIEW-TEMPLATE.md) | Technical | Code quality, architecture, design |
| [SECURITY-REVIEW-TEMPLATE.md](./SECURITY-REVIEW-TEMPLATE.md) | Security | Vulnerability assessment, auth, data handling |
| [PERFORMANCE-REVIEW-TEMPLATE.md](./PERFORMANCE-REVIEW-TEMPLATE.md) | Performance | Latency, throughput, scalability |
| [ACCESSIBILITY-REVIEW-TEMPLATE.md](./ACCESSIBILITY-REVIEW-TEMPLATE.md) | Accessibility | WCAG compliance, user experience |
| [INFRASTRUCTURE-REVIEW-TEMPLATE.md](./INFRASTRUCTURE-REVIEW-TEMPLATE.md) | Infrastructure | Deployment, monitoring, reliability |
| [PROGRESS-REVIEW-TEMPLATE.md](./PROGRESS-REVIEW-TEMPLATE.md) | Progress | Sprint/phase completion, blockers |

## How to Use

1. Choose the appropriate template for your review type
2. Follow the structure and checklist
3. Record findings in the provided sections
4. Track action items in [../review-system/REVIEW-RECOMMENDATIONS-TRACKER.md](../review-system/REVIEW-RECOMMENDATIONS-TRACKER.md)
