## The Framework

A complete, unified approach to reviews covering:
- Technical quality
- Security
- Performance
- Accessibility
- Infrastructure
- Process/progress

**Start here**: [UNIFIED-REVIEW-FRAMEWORK.md](./UNIFIED-REVIEW-FRAMEWORK.md)

## Review Process

1. **Prepare** — Select appropriate [template](../templates/)
2. **Review** — Follow the framework checklist
3. **Document** — Record findings and recommendations
4. **Track** — Add action items to [REVIEW-RECOMMENDATIONS-TRACKER.md](./REVIEW-RECOMMENDATIONS-TRACKER.md)
5. **Follow up** — Update [UNIFIED-REVIEW-TRACKER.md](./UNIFIED-REVIEW-TRACKER.md)

See [REVIEW-PROCESS-GUIDE.md](./REVIEW-PROCESS-GUIDE.md) for details.
