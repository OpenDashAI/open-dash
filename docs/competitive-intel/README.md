## What We Track

- **SERP Rankings** — Keyword rankings and position changes
- **Competitor Domains** — Traffic, backlinks, domain authority
- **Competitor Content** — Blog posts, pricing changes, announcements
- **Social Mentions** — Twitter/X, LinkedIn, industry mentions
- **Market Insights** — Trends, shifts, emerging competitors
