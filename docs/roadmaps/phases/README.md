## Phase Overview

Each phase builds on the previous:

1. **Dashboard Foundation** → YAML-based declarative dashboard
2. **Data Integration** → Dynamic datasource loading from external APIs
3. **User Interface** → Routes, pages, and UI components
4. **Configuration** → Hybrid filesystem + API-based config loading
